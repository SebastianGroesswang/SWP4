Adjecency list: a list of lists, where each inner list contains the neighbors of a vertex. For example, if vertex 0 is connected to vertices 1 and 2, the adjacency list would be [[1, 2], [0], [0]].
Adjency matrix: a 2D array where the entry at row i and column j is 1 if there is an edge from vertex i to vertex j, and 0 otherwise. For example, if vertex 0 is connected to vertices 1 and 2, the adjacency matrix would be:
[[0, 1, 1],
 [1, 0, 0],
 [1, 0, 0]]
